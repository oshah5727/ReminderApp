# Reminder App

## Project Description
The Reminder App is a Java-based application designed to help users track and manage their daily tasks. This project implements a clean architecture, separating the UI from the business logic and data access layers.

## Prerequisites
To run this project, you will need the following installed:
* Java Development Kit (JDK) 11 or higher
* IntelliJ IDEA (Community or Ultimate)
* Git

## How to Run the Project
1. Clone this repository to your local machine: 
   `git clone https://github.com/oshah5727/ReminderApp.git`
2. Open IntelliJ IDEA and select **File > Open**, then navigate to the cloned directory.
3. Allow IntelliJ or Maven/Gradle to download the required dependencies.
4. Locate the main application class (usually containing the `public static void main` method).
5. Click the green **Run** button next to the main class to start the application.
